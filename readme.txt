== Description ==

Widget Taxonomy provides widget for post and custom post type taxonomy display. Taxonomy and Terms disply with listing options and post count of terms.

== Installation ==

Upload the widget-taxonomy  folder to the /wp-content/plugins/ directory. OR add new plugin from WordPress admin. 
Activate the plugin through the ‘Plugins’ menu in WordPress.

e.g.

1. Upload 'widget-taxonomy' to the '/wp-content/plugins/'' directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
